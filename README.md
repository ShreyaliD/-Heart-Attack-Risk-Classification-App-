
# ❤️ Heart Attack Risk Classification App 🩺

🔗 **Live App:** https://heart-attack-risk-classification-harc.streamlit.app/



## 📌 Project Overview

This project is a **Machine Learning-based web application** that predicts whether a person is at **high or low risk of a heart attack** based on medical parameters.

It provides **real-time predictions using a Streamlit interface**.



## 🎯 Objective

* Predict heart attack risk using user input data
* Provide instant results (High Risk / Low Risk)
* Build an interactive and user-friendly web application



## 🧠 Machine Learning Concept

This is a **Binary Classification Problem**, where:

* **0 → Low Risk**
* **1 → High Risk**

The model is trained on a heart disease dataset using machine learning techniques.



## ⚙️ Features

* 🧾 User-friendly input form
* ⚡ Real-time prediction
* 🎯 ML-based classification
* 💡 Clean UI with emojis

## 📊 Input Parameters

* Age
* Resting Blood Pressure
* Cholesterol
* Fasting Blood Sugar
* Maximum Heart Rate
* Oldpeak
* Gender
* Chest Pain Type
* Resting ECG
* Exercise-Induced Angina
* ST Slope



## 🔄 Workflow

1. User enters health details
2. Data preprocessing (encoding + scaling)
3. Model prediction
4. Output displayed:

* 🚨 **High Risk of Heart Attack**
* ✅ **Low Risk of Heart Attack**
